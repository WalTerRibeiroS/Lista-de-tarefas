# 📝To-Do List

Este projeto é uma aplicação simples de lista de tarefas feita para praticar JavaScript, HTML e CSS.
O usuário pode adicionar tarefas, marcar como concluídas e remover tarefas da lista.

## 🚀Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (DOM, eventos, criação dinâmica de elementos)

## 🎯Objetivos

* Praticar manipulação do DOM
* Aprender a criar elementos dinamicamente com JavaScript
* Trabalhar com eventos do teclado e do mouse
* Melhorar a organização de código em projetos pequenos
* Simular funcionalidades usadas em aplicações reais
